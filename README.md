# Telegram / ChatGPT Bot

Telegram-бот с интеграцией ChatGPT, реализованный в рамках учебного проекта.

---

## Возможности

- /random — получение случайного факта
- /gpt — чат с ChatGPT
- /talk — диалог с известной личностью
- /quiz — квиз с подсчётом результатов

---

## Установка

```bash
git clone https://github.com/ridzuki/tg_bot.git
cd telegram-chatgpt-bot
pip install -r requirements.txt
